# public-transportation-guide-updated

```bash
# Clone the repository
git clone https://github.com/ibrahimfnk/public-transportation-guide-updated.git

cd repository

# Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows, use `venv\Scripts\activate`

# Install the required packages
pip install -r requirements.txt

#Run the App
py /app/app.py

```
